help How to Use the AI Prompt Wizard

Welcome to the AI Prompt Wizard! This tool helps you create professional, well-structured prompts for AI assistants. Here's how to make the most of it:

grid_view

Template Cards

Browse and select from pre-built prompt templates designed for common development scenarios:

  • Sprint Planning: Plan your next sprint with AI-powered task breakdown
  • Code Architecture: Get expert architectural recommendations
  • Security Audit: Comprehensive security analysis for your APIs
  • Testing Strategy: Create comprehensive testing strategies
  • And more... Each template is optimized for specific use cases

lightbulb Tip: Click any template card to start the wizard with pre-configured fields.

edit_note

Prompt Builder Form

The wizard guides you through 5 steps to build your perfect prompt:

  1. Project Info: Basic details about your project and tech stack
  2. Context: Background information, team experience, and timeline
  3. Problem: Define the challenge you're facing and constraints
  4. Goals: Specify desired outcomes and success metrics
  5. Generate: Customize output format and generate your prompt

lightbulb Tip: Fields marked with * are required. The more details you provide, the better your AI response will be!

auto_awesome

Generated Prompt Area

Once generated, you can:

  • Copy: Copy the prompt to your clipboard
  • Copy + Open AI: Copy and automatically open your preferred AI platform
  • Edit: Make manual adjustments to the generated prompt
  • Save JSON: Export your prompt configuration for later use

lightbulb Tip: Use the "Fast Mode" toggle to skip the typewriter animation for quicker results.

search

Search & Filters

Quickly find the right template:

  • Search: Type keywords to filter templates by name or description
  • Categories: Filter by Planning, Development, Testing, Security, DevOps, or Documentation
  • Difficulty: Choose Beginner, Intermediate, or Advanced level templates

lightbulb Tip: Use "Clear Filters" to reset all search criteria at once.

save

Draft Saving

Your progress is automatically saved:

  • Per-Template Storage: Each template maintains its own draft independently
  • Auto-Save: When you close the wizard, you'll be prompted to save your progress
  • Resume Anytime: Reopen the same template to continue where you left off

lightbulb Tip: Work on multiple templates simultaneously - each saves its own state!

psychology

Pro Tips for Better Prompts

  • Be Specific: Include exact tech stack versions, team size, and timeline details
  • Add Context: Explain what you've already tried and why it didn't work
  • Define Success: Specify measurable outcomes and acceptance criteria
  • Mention Constraints: Include budget, compliance, or technical limitations upfront
  • Use Power User Mode: Enable advanced options for fine-tuned control over AI responses
  • Try Live Preview: Watch your prompt build in real-time as you type
  • Check Quality Score: Aim for 80%+ quality score for best AI responses
rocket_launch

Common Use Cases

  • Sprint Planning: Break down epics into actionable tasks with effort estimates
  • Code Reviews: Get architectural feedback and best practice recommendations
  • Bug Investigation: Systematically troubleshoot complex issues
  • Documentation: Generate comprehensive technical documentation
  • Performance Optimization: Identify and fix bottlenecks
  • Security Audits: Analyze vulnerabilities and get remediation steps
