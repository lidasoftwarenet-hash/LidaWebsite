12 Powerful Features

Everything you need for professional API debugging

💾

Persistent Mode

Keep all API history across page refreshes and browser restarts. Configure history limits from 100 to 2000 requests.

🔄

Request Replay

One-click replay with full editing. Change method, URL, headers, params, and body. Like Postman built into your browser.

⏱️

Time Range Filtering

DataDog-style time windows: Live, 5min, 10min, 20min, 30min. Focus on recent activity or isolate specific time periods.

📊

Statistics Dashboard

Real-time metrics: success rate, error rate, average response time, fastest/slowest requests, breakdowns by method and status.

💾

Export Filtered Data

Export as JSON, HAR, or CSV. Share with team or import to other tools. Only exports currently filtered data for precision.

📂

Request Grouping

Group by domain, method, or status. Collapsible groups with counts. Perfect for organizing high-volume APIs and microservices.

🪟

Floating Window

Open in separate window, stays on top (optional). Perfect for dual-monitor setups. Monitor APIs while coding.

📑

Tab-Specific Monitoring

Filter to show only current tab's requests. Works in floating windows. Perfect for multi-tab debugging sessions.

🔍

Advanced Multi-Filtering

Combine search, domain, method, status, errors-only, time range, and tab filters. Find exactly what you need instantly.

🚫

Smart Blocklist

Auto-hide tracking, analytics, and ads. Pre-loaded with common domains. One-click block from any request. Custom management.

🎨

Professional UI

VS Code-inspired dark/light themes. Clean, modern, responsive design. Full-width layout with clear visual hierarchy.

📖

Built-in Documentation

Comprehensive help system. Feature explanations with "What it does", "How to use", and "Why it's useful" for every feature.
